# BrewBuddy ☕️

BrewBuddy is a private coffee brewing companion app focused on
pour-over methods like V60, precise timing, and a calm brewing ritual.

## Status
Closed beta – private testing only.

## Access
This project is not publicly available for use.
Access is granted only to invited testers via private links.
